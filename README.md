## About the Project
The Masa Remaja Yearbook Information System is an information system developed for a yearbook vendor named "Masa Remaja." It facilitates the management of yearbook projects, tasks, files, and client interactions, making it easier for the creative team and project managers to collaborate efficiently.

## Features
- **User Management**
    
    Admin can manage user accounts, including creating and assigning roles (Admin, Project Manager, Creative Team, Client).

- **Project Management**

    Admin and Project Managers can create, update, and delete projects.

- **Task Management**

    Project Managers can assign tasks to the Creative Team, and team members can update their task status.

- **File Management**

    Allows uploading, viewing, and managing files related to each project.

- **Client Portal**

    Clients can view project progress, download yearbook files, and provide feedback or testimonials.

- **Public Access**

    Displays a portfolio of completed projects, available services, and client testimonials.

## Tech Stack
- **Backend**: Django
- **Frontend**: Bootstrap
- **Database**: PostgreSQL
- **Version Control**: GitLab
